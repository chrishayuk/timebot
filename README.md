# Install

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v[5:0]u8@10468dc56. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
